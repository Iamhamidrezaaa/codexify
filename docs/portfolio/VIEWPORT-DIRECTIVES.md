# Codexify — Viewport Directives
## Phase 6 · Bridge from Composition Language to Execution

**Status:** Visual direction only. No implementation.  
**Website:** Frozen.  
**Authority:** Composition Language Five (L1–L5) only.

---

# Doctrine

Every important viewport receives a **Design Directive** —  
executable visual law, not a wireframe.

| Term | Meaning |
|------|---------|
| **Directive** | What the eye must do; how weight sits; what is forbidden |
| **Not a directive** | Component names, CSS, grids-as-UI, mock layouts |
| **Composition Language** | Only L1–L5 may be cited as primary systems |
| **Object** | Optional supporting actor unless a Directive explicitly requires one |

### Language key (approved five)

| ID | Name |
|----|------|
| **L1** | Mass against Silence |
| **L2** | Poster Composition |
| **L3** | Monumental Type |
| **L4** | Optical Counterweight |
| **L5** | Compressed Corner |

**L2 is ambient law:** every listed viewport must still pass Poster Test at target score, even when another L# is the primary grammar.

# 01 — Home Hero

| Field | Directive |
|-------|-----------|
| **Purpose** | Establish Codexify in one still: craft, gravity, refusal of SaaS. |
| **Primary emotional outcome** | Quiet certainty — “they understand design.” |
| **Composition Language** | **L1** primary · **L2** mandatory gate · **L3** or **L4** as type mode |
| **Primary gravity** | One optical mass (type *or* material — not both equal) |
| **Secondary gravity** | Single supporting sentence in the silence |
| **Silence ratio** | **62–72%** |
| **Dominant visual mass** | Either monumental statement **or** one material presence — never a device |
| **Object required?** | **No.** Composition is the identity. Material may appear only if it strengthens L1 without stealing L2. |
| **If no — why** | A mandatory Object would revive Phase-4 mascot logic. Hero must survive as pure gravity. |
| **Typography role** | If type is mass → L3. If type is secondary → short support in silence; never CTA-as-gravity. |
| **Poster Test expectation** | Hangable. One idea. No chrome dependence. |
| **Fixation 1** | Primary mass |
| **Fixation 2** | Edge where mass meets silence |
| **Fixation 3** | Supporting line (or micro caption) |
| **Allowed materials** | Paper ground, ink, soft mineral field, restrained archival texture |
| **Forbidden materials** | Glassmorphism, device chrome, gradient blobs, dashboard fragments |
| **Allowed motion** | Opacity settle; micro weight settle after still is complete |
| **Forbidden motion** | Entrance carnivals, parallax showreel, CTA pulse |
| **Typical mistakes** | Split hero; dual CTA; Object competing with type; silence under 50% |
| **Success criteria** | Stranger recalls *air + one weight*, not a feature list. Poster ≥ **23/24**. |

|-------|-----------|

| **Primary emotional outcome** | Immersion + immediate thesis clarity. |
| **Composition Language** | **L1** primary · **L2** mandatory · **L4** for title-vs-field |
| **Primary gravity** | Atmospheric mass (full or large field) **or** monumental case title — one must win |
| **Secondary gravity** | The loser of the above pair (title or field) + one micro meta |
| **Silence ratio** | **40–55%** if atmosphere leads; **58–70%** if title leads |
| **Dominant visual mass** | Industry-true atmosphere (material/light/space) **or** typographic title monument |

| **Typography role** | Title as architecture or as counterweight to atmosphere; never both screaming. |
| **Poster Test expectation** | Case hero must hang as campaign poster for that client world. |
| **Fixation 1** | Winning mass (atmosphere or title) |
| **Fixation 2** | Secondary (the other) |
| **Fixation 3** | Micro meta (industry / year / single clause) |
| **Allowed materials** | Per case dialect (stone, fabric, steel, paper, light…) — one dominant |
| **Forbidden materials** | Device mockups as hero; before/after sliders; logo salad |
| **Allowed motion** | Atmospheric settle; slow exposure; still-first |
| **Forbidden motion** | Showreel autoplay as composition crutch |
| **Typical mistakes** | Template case hero; title + mockup split; dual CTA |
| **Success criteria** | Industry dialect unmistakable in one still. Poster ≥ **23/24**. |

# Cross-viewport laws

1. **L2 never sleeps.** If it fails Poster Test, the Directive failed — regardless of L#.  
2. **One primary gravity per viewport.** Secondary may exist; tertiary is usually a mistake.  
3. **Object never rescues weak composition.**  
4. **CTA is never primary gravity** on 01–08. Contact channel may be secondary gravity on 09 only.  
5. **RTL:** fixation order follows optical weight first, script direction second.  
6. **Dialect over decoration:** case materials may change; Language IDs do not.  
7. **Footer cannot out-poster the Hero.**
