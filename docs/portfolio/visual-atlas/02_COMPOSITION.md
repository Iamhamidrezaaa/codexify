# 02 — Composition
## Visual Atlas

Compositions are gravity systems.  
Classify first; reference second.

---

## Pattern catalog

### P01 — Single Object
**When it works:** Object has enough mass, texture, and identity to own the viewport.  
**When it fails:** Object is weak, small, or decorative; silence becomes unfinished.  
**Industries:** Luxury, healthcare, architecture, fashion (quiet).

### P02 — Object + Typography
**When it works:** Clear primary/secondary roles; type locks to object edge or orbits.  
**When it fails:** Type and object share the same optical center with equal weight.  
**Industries:** All — default studio dialogue.

### P03 — Object + Caption
**When it works:** Museum seriousness; caption confirms craft.  
**When it fails:** Caption becomes a paragraph competing with the object.  
**Industries:** Culture, architecture, luxury, healthcare.

### P04 — Object + Empty Space
**When it works:** Pedestal silence; proportions intentional.  
**When it fails:** Leftover whitespace with no shaped void.  
**Industries:** Luxury, architecture, studio manifesto.

### P05 — Split Mass
**When it works:** Two masses in dialogue (image | type) with a hard seam.  
**When it fails:** 50/50 without hierarchy; looks like a template split.  
**Industries:** Fashion, editorial, case study openings.

### P06 — Floating Mass
**When it works:** Object lifts with contact shadow and controlled isolation.  
**When it fails:** Fake float with glow; or object lost in void.  
**Industries:** Technology (careful), luxury objects, product craft.

### P07 — Monument
**When it works:** Near-full-bleed object/type; ceremonial scale.  
**When it fails:** Monumental emptiness (big type, no object).  
**Industries:** Luxury, architecture, culture.

### P08 — Edge Crop
**When it works:** Crop creates tension and modernity; implies more world off-frame.  
**When it fails:** Accidental clipping; faces/critical edges butchered without intent.  
**Industries:** Fashion, photography, editorial.

### P09 — Diagonal Gravity
**When it works:** Dynamic axis without chaos; still readable.  
**When it fails:** Scattered angles; carnival.  
**Industries:** Experimental campaigns, select fashion — rare for Codexify core.

### P10 — Poster Layout
**When it works:** Still image survives as print poster; one idea.  
**When it fails:** Too many messages for one poster.  
**Industries:** All brand heroes.

### P11 — Gallery Layout
**When it works:** Multiple objects as curated sequence with rhythm.  
**When it fails:** Equal tiles; Instagram grid energy.  
**Industries:** Portfolio indexes, culture.

### P12 — Architectural Grid
**When it works:** Structure visible as beauty; measured alignments.  
**When it fails:** Grid wallpaper; decoration without measurement.  
**Industries:** Architecture, interior, education, finance (order).

### P13 — Object + Grid
**When it works:** Object sits in measured field; grid supports, doesn’t compete.  
**When it fails:** Grid denser than object.  
**Industries:** Architecture, process, systems chapters.

### P14 — Object + Shadow
**When it works:** Light logic proves the object exists in space.  
**When it fails:** Drop-shadow stickers; inconsistent light.  
**Industries:** Luxury, product, hospitality night.

### P15 — Object + Layer
**When it works:** Archive/time implied (sheets, veils, planes).  
**When it fails:** Too many layers; muddy depth.  
**Industries:** Studio, process, paper-led brands.

### P16 — Object + Crop + Caption
**When it works:** Modern editorial completeness.  
**When it fails:** Over-labeled; caption fights crop.  
**Industries:** Fashion, culture, case figures.

### P17 — Typographic Object
**When it works:** Type has material mass; ground still material.  
**When it fails:** Big type on empty canvas wireframe.  
**Industries:** Studio identity, manifesto, Swiss-leaning moments.

### P18 — Threshold Composition
**When it works:** Arrival feeling; left/right or before/through.  
**When it fails:** Literal doorway clichés.  
**Industries:** Hospitality, architecture, healthcare entry.

### P19 — Dual Object Dialogue
**When it works:** Two materials converse (steel|fabric, stone|paper).  
**When it fails:** Unrelated objects collage.  
**Industries:** Interior, fashion, hospitality.

### P20 — Index Strip + Hero Object
**When it works:** Navigation of chapters without killing gravity.  
**When it fails:** Index becomes visual noise.  
**Industries:** Case studies, monographs.

## Failure signatures (memorize)

1. **Wireframe hero** — type + lines, no object  
2. **Template split** — 50/50 without hierarchy  
3. **Grid vanity** — structure louder than content  
4. **Caption novel** — text mass competing with object  
5. **Glow float** — fake premium  
6. **Equal gallery** — no curation rhythm
