## Example assembly (worked)

**Need:** Healthcare clinic case hero  

1. Bible dialect: Healthcare  
2. Object: Fabric (primary) + Light (secondary) → see OBJ-05, OBJ-07  
3. Composition: Object + Empty Space (P04) or X23  
4. Type: TP-H Care Readable statement + TP-B caption  
5. Motion: M03 Breath opacity (optional), never M06  
6. Poster Test: must score ≥20 still  
7. Nav: N04 invisible until scroll + N15 contextual atmosphere  

**Need:** Luxury horology case hero  

1. Object: Steel / Mechanical  
2. Composition: Single Object or Monument  
3. Type: TP-G  
4. Motion: M06  
5. Forbidden: fabric spa cues; bright medical sage as primary

## Conflict resolver

If two combinations disagree:

1. Industry dialect wins over aesthetic preference  
2. Object truth wins over typography preference  
3. Poster Test wins over motion preference  
4. Creative Bible commandments win over Atlas taste
