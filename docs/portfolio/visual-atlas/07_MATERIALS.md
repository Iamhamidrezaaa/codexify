# 07 — Materials
## Visual Atlas

Materials are Object flesh.  
Study characteristics — not stock texture packs.

---

## Paper

**Visual characteristics:** Fiber, soft edge, ink absorbency, warm bias  
**Lighting:** Diffuse; avoid wet gloss  
**Texture:** Subtle tooth; visible at crop  
**Industries:** Studio, education, culture, process  
**Forbidden:** Torn scrapbook chaos; coffee-stain gags  

## Stone

**Visual characteristics:** Matte mineral; porosity; cool/warm grey  
**Lighting:** Raking soft light  
**Texture:** Fine grain, not rocky cartoon  
**Industries:** Architecture, interior, hospitality quiet  
**Forbidden:** Fantasy ruins; shiny vanity marble without concept  

## Steel

**Visual characteristics:** Brush direction; hard edge; controlled specular  
**Lighting:** Long highlight; often dark ground  
**Texture:** Linear brush, not mirror disco  
**Industries:** Luxury objects, precision tech  
**Forbidden:** Neon chrome; cyber steel  

## Glass

**Visual characteristics:** Edge light; clean refraction hint  
**Lighting:** Rim; avoid rainbow flares  
**Texture:** Smooth; dirt only if conceptual  
**Industries:** Retail calm, hospitality, select tech  
**Forbidden:** Glassmorphism UI panels  

## Fabric

**Visual characteristics:** Weave, drape, fold shadow  
**Lighting:** Soft wrap  
**Texture:** Matte textile  
**Industries:** Healthcare, fashion, hospitality  
**Forbidden:** Spa stock towels as identity  

## Concrete

**Visual characteristics:** Aggregate hint; board-form; cool mass  
**Lighting:** Soft or slit light  
**Texture:** Mineral roughness controlled  
**Industries:** Architecture, culture, select brutal moments  
**Forbidden:** Grunge overlays; “urban decay” cosplay on care brands  

## Wood

**Visual characteristics:** Grain river; joinery; cut ends  
**Lighting:** Warm directional  
**Texture:** Natural variation  
**Industries:** Interior, architecture, hospitality  
**Forbidden:** Fake laminate shine  

## Ink

**Visual characteristics:** Density variation; letterpress bite; bleed control  
**Lighting:** Flat or slight raking on emboss  
**Texture:** On paper only  
**Industries:** Editorial, studio, print-led  
**Forbidden:** Ink splat decoration  

## Emboss

**Visual characteristics:** Raised form; highlight on ridge  
**Lighting:** Low angle essential  
**Texture:** Paper deformation  
**Industries:** Stationery, luxury print, studio craft  
**Forbidden:** Fake CSS bevel as brand substance without real logic  

## Deboss

**Visual characteristics:** Recessed impression; soft inner shadow  
**Lighting:** Soft top light  
**Texture:** Compressed fiber  
**Industries:** Covers, cards, quiet luxury  
**Forbidden:** Overused on every card UI  

## Hairlines

**Visual characteristics:** Near-invisible rules; registration  
**Lighting:** N/A — graphic material  
**Texture:** None — precision  
**Industries:** All editorial systems  
**Forbidden:** Hairlines replacing objects; decorative box frames everywhere  

## Registration marks

**Visual characteristics:** Crop marks, crosshairs, print meta  
**Lighting:** N/A  
**Texture:** None  
**Industries:** Studio meta, print-aware heroes (subtle)  
**Forbidden:** Loud printer UI cosplay; marks as clutter  

## Architectural shadows

**Visual characteristics:** Contact shadow; cast geometry; soft ambient  
**Lighting:** Must match object story  
**Texture:** Softness communicates distance  
**Industries:** All object photography / plates  
**Forbidden:** Glow; colored neon shadows; inconsistent multi-sun

## Material pairing matrix (allowed defaults)

| Primary | Good secondary |
|---------|----------------|
| Paper | Ink, hairlines, emboss |
| Stone | Light, architectural shadow |
| Steel | Shadow, mechanical |
| Fabric | Light, soft shadow |
| Wood | Soft light, fabric |
| Glass | Liquid, light edge |
| Concrete | Light slit, shadow |
