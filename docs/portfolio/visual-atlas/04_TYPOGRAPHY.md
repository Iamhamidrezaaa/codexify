# 04 — Typography
## Visual Atlas

Typography is architecture (Creative Bible §08).  
References teach hierarchy, not font shopping.

---

## Principles distilled from references

### Display scale
Display must commit. Mid-size “almost big” is the enemy.  
When display is the Object, material ground still exists.

### Hierarchy
One primary voice per viewport.  
Caption and overline are craft layers, not alternate headlines.

### Whitespace
Space between lines and around blocks is structural.  
Tight leading for night luxury names; open leading for care reading.

### Alignment
Shared edges with Objects.  
Captions lock to plate edges or deliberate offset rails.

### Caption systems
Museum labels: small, precise, rarely wrap into novels.  
Figure IDs behave like inventory marks.

### Numerals
Tabular where measurement matters.  
Monumental numerals only when they are Objects (indexes, chapters).

### Editorial rhythm
Alternate long reading measures with caption bursts.  
Avoid walls of equal paragraphs.

## Typography patterns for Codexify

| Pattern | Use |
|---------|-----|
| TP-A Monument Statement | Heroes when type shares gravity with object |
| TP-B Museum Caption | Figures, plates, archival notes |
| TP-C Chapter Overline | Section identity without competing |
| TP-D Measure Essay | Process, studio, long case prose |
| TP-E Index Numerals | Sticky indexes, chapter rails |
| TP-F Edge-locked Title | Type welded to object edge |
| TP-G Sparse Luxury Name | Night luxury product names |
| TP-H Care Readable | Healthcare dialect body |

### Never

- Decorative script as primary studio voice  
- All-caps paragraphs  
- Mid-hierarchy soup (five sizes, no leader)  
- Colored body text  
- Type as filler for missing objects
