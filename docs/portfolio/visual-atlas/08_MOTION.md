# 08 — Motion
## Visual Atlas

Do not describe animation curves here.  
Describe **intention** — why motion exists and how it serves the Object.

---

## Motion law

Motion amplifies Objects.  
If the still fails the Poster Test, motion will not save it.

---

## Thirty motion intentions

### M01 — Object arrival
**Why:** Announce gravity  
**Supports Object:** Reveals mass into silence  
**Succeeds when:** Object feels inevitable afterward  

### M02 — Paper lift
**Why:** Suggest archive/time  
**Supports Object:** Paper planes gain physics  
**Succeeds when:** Feels tactile, not UI accordion  

### M03 — Breath opacity
**Why:** Care dialects; lower pulse  
**Supports Object:** Soft materials appear gently  
**Succeeds when:** Invisible if you aren’t looking for it  

### M04 — Light creep
**Why:** Architecture/hospitality atmosphere  
**Supports Object:** Light-as-object becomes alive  
**Succeeds when:** Slow like sun, not flashlight  

### M05 — Mask reveal
**Why:** Controlled disclosure  
**Supports Object:** Crop/aperture logic  
**Succeeds when:** Geometry feels architectural  

### M06 — Mechanism settle
**Why:** Precision trust  
**Supports Object:** Steel/mechanical  
**Succeeds when:** Short, exact, done  

### M07 — Typographic settle
**Why:** Statement lands after object  
**Supports Object:** Hierarchy sequence  
**Succeeds when:** Type never arrives before object without reason  

### M08 — Caption delay
**Why:** Museum pacing  
**Supports Object:** Object first, scholarship second  
**Succeeds when:** Caption doesn’t race  

### M09 — Chapter crossfade
**Why:** Monograph page turn  
**Supports Object:** Continuity between plates  
**Succeeds when:** Still readable mid-transition  

### M10 — Index numeral morph
**Why:** Orientation while walking gallery  
**Supports Object:** Indirect — supports reading of sequence  
**Succeeds when:** One signature only  

### M11 — Sticky label hold
**Why:** Reading while context changes  
**Supports Object:** Keeps chapter identity  
**Succeeds when:** Doesn’t cover object mass  

### M12 — Parallax refusal
**Why:** Intentionally almost none  
**Supports Object:** Prevents cheap depth  
**Succeeds when:** Space feels stable  

### M13 — Scroll-as-walk
**Why:** Architecture/hospitality journey  
**Supports Object:** Thresholds appear in sequence  
**Succeeds when:** Pace matches room scale  

### M14 — Soft height expand
**Why:** Progressive disclosure in care  
**Supports Object:** Content without jump scare  
**Succeeds when:** No playful bounce  

### M15 — Focus ring calm
**Why:** Access as craft  
**Supports Object:** Interaction hygiene  
**Succeeds when:** Visible and non-theatrical  

### M16 — Hero dissolve on leave
**Why:** Exit without drama  
**Supports Object:** Releases gravity for next chapter  
**Succeeds when:** Doesn’t feel like a magic trick  

### M17 — Material rake (light move)
**Why:** Show texture truth  
**Supports Object:** Stone/steel/fabric grain  
**Succeeds when:** Subtle highlight travel  

### M18 — Shadow lengthen
**Why:** Time of day emotion  
**Supports Object:** Shadow category  
**Succeeds when:** Slow, single source logic  

### M19 — Layer peel
**Why:** Process thinking visible  
**Supports Object:** Paper layers  
**Succeeds when:** Finite layers, not infinite  

### M20 — Still-first documentation
**Why:** Motion designed from posters  
**Supports Object:** Ensures poster survival  
**Succeeds when:** Keyframes alone are beautiful  

### M21 — Reduced-motion equivalence
**Why:** Ethics + craft  
**Supports Object:** Meaning without motion  
**Succeeds when:** No locked content  

### M22 — Hover restraint
**Why:** Desktop affordance without play  
**Supports Object:** Slight density/opacity only  
**Succeeds when:** No scale pop carnival  

### M23 — Route atmosphere switch
**Why:** Dialect continuity  
**Supports Object:** Chrome respects object field  
**Succeeds when:** Instantly readable  

### M24 — Progress hairline
**Why:** Reading tool  
**Supports Object:** Indirect support  
**Succeeds when:** Almost invisible  

### M25 — Gallery snap calm
**Why:** Plate sequence  
**Supports Object:** Each plate remains poster  
**Succeeds when:** No aggressive magnetic fighting  

### M26 — Type mask by object edge
**Why:** Object/type relationship  
**Supports Object:** Type respects mass  
**Succeeds when:** Legibility preserved  

### M27 — Exit hold (end pause)
**Why:** Let last object breathe  
**Supports Object:** Memory formation  
**Succeeds when:** User isn’t rushed to CTA  

### M28 — Contact shadow fade-in
**Why:** Prove object rests in world  
**Supports Object:** Floating mass honesty  
**Succeeds when:** No glow  

### M29 — Industry dialect tempo
**Why:** Care slow / steel exact / architecture slow walk  
**Supports Object:** Emotion match  
**Succeeds when:** Tempo chosen before effects  

### M30 — One memorable idea per experience
**Why:** Signature without circus  
**Supports Object:** Attention economy of craft  
**Succeeds when:** Other motions stay subordinate  

---

## Motion rejection list

- Bounce, elastic, confetti  
- Motion used to hide empty composition  
- Autoplaying spectacle in heroes  
- Multiple signatures on one page  
- Essential meaning only available in motion
