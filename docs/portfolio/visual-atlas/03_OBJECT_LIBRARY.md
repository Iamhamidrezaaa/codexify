## Visual Atlas · Object System

## How to specify an Object

Every Object brief must state:

Emotion · Industries · Visual characteristics · Scale · Lighting · Cropping · Motion suitability · Typography compatibility · Forbidden combinations · References

---

## OBJ-01 Stone

| Field | Definition |
|-------|------------|
| Emotion | Permanence, patience, quiet strength |
| Industries | Architecture, interior, hospitality (quiet), education (heritage), culture |
| Visual characteristics | Matte mineral body; soft fracture; cool/warm grey; minimal specular |
| Scale | Medium→monument; micro stone only as texture support |
| Lighting | Raking soft light to reveal porosity; avoid shiny wet look |
| Cropping | Large planes; edge of break; avoid pebble scatter |
| Motion suitability | Very slow; heavy settles |
| Typography compatibility | Strong with charcoal ink; captions archival |
| Forbidden | Cartoon rocks; fantasy ruins; glossy marble vanity without concept |
| References | Zumthor interiors; Ando concrete-as-stone cousin; Norm material plates |

## Forbidden combinations (global)

| Pair | Why forbidden |
|------|----------------|
| Steel + playful confetti | Emotion clash |
| Fabric care + neon tech glow | Dialect collapse |
| Paper editorial + dashboard UI chrome | Category error |
| Organic + before/after clinical shock | Ethics + brand harm |
| Architecture void + stock handshake | Meaning destruction |
| Mechanical macro + lifestyle yacht | Luxury vulgarity |

---

## Object selection protocol

1. Name industry dialect.  
2. Choose primary Object category.  
3. Choose optional secondary (material support only).  
4. Check forbidden combinations.  
5. Bind to composition pattern + poster test.
