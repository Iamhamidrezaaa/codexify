# 05 — Navigation
## Visual Atlas

Navigation is secondary gravity.  
It must never become the Object — unless the product *is* navigation (not our case).

---

## Interaction philosophy (studio law)

- Visible enough to trust; quiet enough to respect the Object  
- Presence can change with atmosphere (light paper vs night salon)  
- Transitions are settles, not tricks  
- Hierarchy: primary routes few; secondary discoverable  
- Behavior predicts location: where I am / where from / where next  

---

## Twenty premium navigation systems

### N01 — Museum persistent quiet bar
**Behavior:** Always available, low contrast until need  
**Visibility:** High accessibility, low visual weight  
**Transitions:** Instant or soft fade  
**Hierarchy:** Few items  
**Philosophy:** Institution over marketing  
**Borrow:** Calm permanence  
**Never copy:** Buried hamburger-only on desktop without reason

### N02 — Monograph contents rail
**Behavior:** Chapter jump index  
**Visibility:** Desktop side/top subtle numerals  
**Transitions:** Scroll-linked current state  
**Hierarchy:** Chapter > page  
**Philosophy:** Book logic  
**Borrow:** Reading progress + chapter identity  
**Never copy:** Dot fireworks

### N03 — Exhibition sticky index
**Behavior:** Active project/numeral updates with scroll  
**Visibility:** Large watermark or quiet list  
**Philosophy:** Gallery walk  
**Borrow:** One memorable index behavior  
**Never copy:** Multiple competing signatures

### N04 — Invisible until scroll
**Behavior:** Transparent over hero object; solidifies on scroll  
**Visibility:** Conditional  
**Philosophy:** Object first  
**Borrow:** Atmosphere-aware chrome  
**Never copy:** Unreadable nav on busy photography

### N05 — Footer as secondary map
**Behavior:** Deep links live in closing  
**Visibility:** End of journey  
**Philosophy:** Colophon + map  
**Borrow:** Reduce header load  
**Never copy:** Mega-footer marketplace

### N06 — Text-as-nav (rare archives)
**Behavior:** Find rather than browse  
**Philosophy:** Collection scale  
**Borrow:** Only if collection demands  
**Never copy:** Search replacing IA for a small studio site

### N07 — Horizontal chapter ticker
**Behavior:** Soft scrub of sections  
**Philosophy:** Filmstrip chapters  
**Borrow:** Case study chaptering  
**Never copy:** Marquee as primary IA

### N08 — Breadcrumb monograph path
**Behavior:** Work / Project / Chapter  
**Philosophy:** Orientation without noise  
**Borrow:** Case study orientation  
**Never copy:** Overlong crumbs

### N09 — Prev/Next artifact nav
**Behavior:** Adjacent projects  
**Philosophy:** Gallery continuum  
**Borrow:** Case study endings  
**Never copy:** Fake infinite related content

### N10 — Minimal wordmark-only header
**Behavior:** Home is the only persistent  
**Philosophy:** Extreme confidence  
**Borrow:** Rare ceremonial pages  
**Never copy:** As default entire site without IA

### N11 — Split primary / utility
**Behavior:** Works+Studio vs Contact utility  
**Philosophy:** Clear jobs  
**Borrow:** Reduce equal-weight link soup  
**Never copy:** Utility styled as primary spectacle

### N12 — Anchor essay progress
**Behavior:** Long-form section tracking  
**Philosophy:** Reading tool  
**Borrow:** Process/studio essays  
**Never copy:** Fake progress for short pages

### N13 — Overlay menu as room
**Behavior:** Full-field menu with object field behind dimmed  
**Philosophy:** Menu as designed space  
**Borrow:** Mobile/desktop ceremonial open  
**Never copy:** Template burger panels

### N14 — Language/script switch quiet
**Behavior:** FA/EN if ever needed  
**Philosophy:** Respect without flags circus  
**Borrow:** Subtlety  
**Never copy:** Flag icons as decoration

### N15 — Contextual case chrome
**Behavior:** Inside case study, nav adapts to theme atmosphere  
**Philosophy:** Dialect continuity  
**Borrow:** Dark/light heroes  
**Never copy:** Losing global orientation

### N16 — Index page as nav object
**Behavior:** /work is the map  
**Philosophy:** Exhibition entrance  
**Borrow:** Portfolio as place  
**Never copy:** Card grid marketplace

### N17 — Soft underline current
**Behavior:** Current route marked  
**Philosophy:** Orientation honesty  
**Borrow:** Always  
**Never copy:** Loud pill “active” chips

### N18 — Contact as action not page theater
**Behavior:** Mail/conversation pathway  
**Philosophy:** Reduce dead ends  
**Borrow:** Honest CTAs  
**Never copy:** Fake multi-step funnels in header

### N19 — Keyboardable quiet focus
**Behavior:** Focus rings respectful  
**Philosophy:** Access is craft  
**Borrow:** Non-negotiable  
**Never copy:** Removing focus for “beauty”

### N20 — Reduced-motion nav parity
**Behavior:** Same IA without animation dependency  
**Philosophy:** Motion amplifies, never gates  
**Borrow:** Law  
**Never copy:** Essential info only in motion

---

## Navigation anti-patterns

- More than ~5 equal primary links without hierarchy  
- Nav competing with hero Object  
- Unreadable contrast on material fields  
- Playful hover that breaks editorial tone  
- Dead routes  
- Missing “where I am”
