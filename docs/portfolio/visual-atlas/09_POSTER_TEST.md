# 09 — Poster Test
## Visual Atlas · Evaluation Law

Every Hero.  
Every Case Study opening.  
Every major section.  

Must survive as a **still poster**.

If it needs motion, scroll, or hover to feel premium, it fails.

---

## Protocol

1. Capture the viewport (or designed frame) as a still image.  
2. Hide browser UI mentally (or literally crop chrome).  
3. View at arm’s length on phone and at poster distance on desktop.  
4. Score with the criteria below.  
5. Fail any score under threshold — revise Object/composition before implementation debates.

---

## Objective criteria (score 0–2 each)

| # | Criterion | 0 | 1 | 2 |
|---|-----------|---|---|---|
| 1 | **Dominant Object present** | None / chrome only | Weak / ambiguous | Clear named Object |
| 2 | **Gravity** | Eye wanders | Competing centers | One sun |
| 3 | **Silence quality** | Empty leftover | Partial pedestal | Shaped intentional void |
| 4 | **Hierarchy** | Unreadable / flat | Partial | Instant primary→secondary |
| 5 | **Material truth** | Generic / stocky | Suggestive | Believable material language |
| 6 | **Industry dialect** | Wrong costume | Mild mismatch | Correct atmosphere |
| 7 | **Typography architecture** | Soup / wireframe type | Adequate | Load-bearing, edged |
| 8 | **Caption / craft micro** | Missing when needed / noisy | Present | Precise museum craft |
| 9 | **Crop / edge intention** | Accidental | Partial | Decisive |
| 10 | **Memorability** | Forget in 10s | Mild recall | Poster recall |
| 11 | **Anti-template** | Feels template | Borderline | Authored |
| 12 | **Bible compliance** | Breaks commandments | Minor tension | Aligned |

**Maximum:** 24  
**Pass threshold:** 18  
**Hero threshold:** 20  
**Ship-blocking fails:** Criterion 1 = 0, or Criterion 2 = 0, or Criterion 12 = 0

---

## Binary kill switches (instant fail)

- No dominant Object  
- Device mockup as the idea  
- Glassmorphism / glow / neon gradient spectacle  
- Stock emotion theater  
- Unreadable navigation on the field  
- Motion required to understand the message
