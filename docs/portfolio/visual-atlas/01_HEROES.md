## Hero borrowing checklist

Before designing any Codexify hero, confirm:

1. Named Object category from Object Library  
2. Composition pattern from Composition Atlas  
3. Poster Test pass (§09)  
4. Industry dialect match (Bible §06)  
5. At least one “never copy” respected from related refs
