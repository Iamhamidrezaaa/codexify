---

## Brand Name

**Sora Residence**

## Industry

Architecture studio — residential and small cultural projects.

## Brand Personality

Contemplative. Structural. Light-aware. Unhurried. Honest about material.

## Target Audience

- Private clients commissioning homes
- Cultural institutions seeking a measured partner
- Peers and press who read plans as carefully as photographs

## Business Goal

Present built work with spatial intelligence; make process visible without turning the site into a PDF dump; attract briefs that value silence over spectacle.

## Creative Direction

Paper, concrete, sky. The interface should feel like walking a corridor — thresholds, pauses, aligned axes.

### Color Palette

| Role | Direction |
|------|-----------|
| Ground | Warm paper `#F3F0EA` |
| Ink | Soft black `#1A1A1A` |
| Line | Pencil grey `#B7B2A8` |
| Accent | Oxidized copper / muted clay `#9C6B4A` |
| Sky void | Pale blue-grey `#D9E0E4` for large empty fields |

### Typography Direction

- Display: architectural grotesque or humanistic sans — wide, calm
- Captions: small, mono-adjacent for drawing references
- Project titles: one line when possible; never stacked marketing slogans

### Photography Direction

- Architecture photography: dawn/dusk, empty rooms, material close-ups
- Occasional plan or elevation as graphic — not decoration
- People rare; if present, scaled small in space

### Layout Direction

- Strong horizontal datums
- Full-bleed plates alternating with narrow caption columns
- Project index like a contents page of a monograph

### Interaction Direction

- Scroll as walking sequence
- Sticky project meta while images pass (sparingly)
- Image reveals that feel like lifting a sheet of tracing paper

### Motion Direction

- Slow. Linear. No elastic
- Opacity and soft vertical drift only
- Prefer mask geometry over scale pops

### UI Personality

Monograph. Drawing set. Museum wall labels.

### Voice & Tone

Precise. Spatial. Short sentences. Describes light and proportion before amenities.

### Keywords

residence, threshold, plan, concrete, aperture, silence, monograph, site

## Visual Direction Summary

**Material world:** paper, timber, plaster, sky.  
**Composition:** axis and void.  
**Codexify note:** Lightest ground in the six — proof we can do quiet day as well as noir.
