# Codexify — Object Design Lab
## Phase 4 · Physical Identity (Website Frozen)

**Status:** Concept archive only. No implementation.  
**Bound by:** Creative Bible §03 · Visual Atlas Object Library  
**Law:** If removing it does not change meaning, it was never an Object.

This document designs the **signature Object** of Codexify as a studio —  
the thing people remember after the browser closes.  
Not a layout. Not a UI. Not a moodboard shape.

---

# Method

Twenty Objects across eight categories.  
Each must survive as:

1. A thing a 3D artist could model without inventing missing structure  
2. A still poster without motion crutches  
3. A studio thesis (editorial craft, material truth, intentional silence)  
4. Non-decoration (removal collapses meaning)

**Forbidden forms discarded at ideation:** rectangles-as-cards, floating glass UI, devices, dashboards, gradient blobs, random “sculpture,” AI cliché monoliths with soft bevels.

## 01 — Datum Cleat
**Category:** Structural · Industrial

| Field | Spec |
|-------|------|
| **Name** | Datum Cleat |
| **Concept** | A precision alignment cleat — the physical act of “bringing things into register.” |
| **Material** | Brushed tool steel + matte black oxide bolt |
| **Emotional role** | Exactness without cold arrogance |
| **Physical description** | A short, thick cleat with a V-groove along its length; one countersunk bolt; chamfered ends; no logo. |
| **Lighting** | Hard raking light along the brush grain; single specular strip |
| **Scale** | Hand-held monument (~18 cm long) → fills hero as macro |
| **Weight** | Dense; feels like it could hold a door true |
| **Texture** | Linear brush + oxide mute on bolt head |
| **Suitable industries** | Architecture, interior, luxury hardware, selective tech |
| **Poster suitability** | High — silhouette + groove = instant icon |
| **Motion suitability** | Bolt settles a quarter-turn; groove catches light |
| **Why memorable** | The V-groove is a signature cut — readable at thumbnail |
| **Why Codexify** | Design as registration: nothing drifts |
| **Why not decoration** | It *is* the argument: alignment is the craft |

**Narrative.**  
Codexify does not invent atmospheres; it brings edges into true. The Datum Cleat is that ethics made metal: a small, heavy instrument whose only job is to refuse misalignment.

**3D brief.**  
Model an elongated steel body (~180×42×28 mm) with a continuous 90° V-groove machined along the top face, 1 mm edge fillets, longitudinal brush texture (Ra ~0.8). One M8 socket bolt in black oxide near one end, countersunk. Soft contact shadow. No branding engravings except optional micro “Δ” stamp 0.2 mm deep.

**Client memory.**  
They leave remembering “the metal with the groove” — not a headline. That groove becomes shorthand for “this studio measures.”

# Recommendation — ONE Signature Object

## **Letterpress Chase** (Object 03)

### Why this is the Codexify signature

1. **Ownable silhouette** — An open iron frame is as identifiable as a swoosh once repeated: mass around a composed void.  
2. **Silence is structural** — The empty center is not leftover whitespace; it is the chase window. Bible law made metal.  
3. **Editorial without flat paper** — Keeps publishing DNA; escapes the forbidden “rectangle sheet as hero” trap.  
4. **Gravity** — Cast iron reads heavy in stills; Poster Test native.  
5. **Not a website object** — It belongs on a desk, in a film still, on a monograph cover, in a 3D turntable — then, later, on a site.  
6. **Anti-SaaS** — No device, no card, no dashboard. A press instrument.

### How it should live (design intent only — no build)

- Always show **furniture + type fragment** inside — never an empty decorative frame.  
- Ground: warm paper or dark workshop — never UI grey.  
- Typography orbits outside the chase or locks to the outer rail — never fights the window.  
- Motion only as **pressure**: quoin turns, type settles. Still must already work.  
- Optional secondary Object later: Archival Screw-Post as “system sibling,” not rival.

### What we explicitly do *not* choose as #1

- **Screw-Post** — superb, recommend as secondary signature in the Object family.  
- **Datum Cleat** — keep for architecture/luxury case dialects.  
- **Phase 3 Archival Folio** — valuable exploration; not the long-term coat of arms (too close to flat plane / rectangle risk).
