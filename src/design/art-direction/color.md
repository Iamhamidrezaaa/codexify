# Color Rules

Single accent. No gradients. No decorative color fields.

| Semantic | Token | Use |
|----------|-------|-----|
| Canvas | `--bg` | Page background |
| Ink | `--fg` | Primary text |
| Muted | `--fg-muted` | Secondary text |
| Border | `--border` | Dividers |
| Accent | `--accent` | Indexes, underlines, focus |

Accent is rare. If everything is accented, nothing is.
