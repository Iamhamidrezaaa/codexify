# Grid & Composition Rules

- **12 columns** desktop · 8 tablet · 4 mobile
- Gutters / margins from `--gutter-*` and `--margin-*`
- Max width: `--container-wide`
- Component: `<Grid />` in `@/components/layout/Grid`

## Composition

Asymmetry is planned (e.g. headline spans start columns; meta sits at the end).  
Never center-everything template layouts.  
Whitespace is a primary material.
