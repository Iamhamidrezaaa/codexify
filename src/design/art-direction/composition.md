# Composition Rules

- One job per section: one purpose, one headline, one supporting line when needed
- Typography is the primary visual plane — not decoration behind content
- Prefer large silence over decorative chrome
- Asymmetry must map to the grid (see [grid.md](./grid.md))
- Avoid card stacks, pill clusters, and stat strips
- The first viewport should read as one editorial composition, not a dashboard

Images (when used): architecture, geometry, negative space — never stock or AI filler.
