# Interaction001 — Magnetic Text

## When to use
Quiet life on short titles. Cursor feedback without a custom cursor.

## When NOT to use
Body copy, forms, long paragraphs.

## Performance
Cost: low. Prefer strings under 40 characters.

## Accessibility
Honors `prefers-reduced-motion`. Screen readers get the full label.

## Limitations
DOM nodes scale with character count.

## Future
Canvas rasterization for longer strings.
